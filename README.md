# AutoLine – Web Application pentru Programări la Auto service

## Descriere
**AutoLine** este o aplicație web dezvoltată pentru gestionarea programărilor la un service auto. Scopul acesteia este de a simplifica procesul de programare și de a îmbunătăți organizarea activităților într-un service.

Aplicația permite utilizatorilor să creeze, vizualizeze și gestioneze programări direct din browser.

---

## Funcționalități
- Crearea unei programări auto
- Salvarea datelor în LocalStorage
- Vizualizarea programărilor existente
- Ștergerea programărilor
- Panou Admin pentru vizualizare completă
- Interfață simplă și intuitivă

---

## Tehnologii utilizate
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (pentru stocarea datelor)
